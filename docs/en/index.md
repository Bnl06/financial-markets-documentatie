## How to use this site

1. For each unit, first read the **key idea** and the **exam focus**.
2. Do not learn definitions in isolation; always connect them to an example.
3. Practise open questions aloud: definition → mechanism → example → consequence.
4. Redo the calculation exercises without looking at the solution.
5. Right before the exam, repeat the glossary and the formula sheet.

!!! tip "Open questions"

    In open questions you earn points for structure. Do not start with random details. Start with a short definition, then explain the mechanism, give an example and end with the impact on financial markets or institutions.
