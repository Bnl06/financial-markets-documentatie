## How to use this site

1. For each unit, first read the **core sentence** and the **exam focus**.
2. Do not learn definitions separately; always learn them with an example.
3. Practise open questions out loud: definition → mechanism → example → consequence.
4. Redo the numerical exercises without looking at the solution.
5. Review the glossary and formula sheet right before the exam.

!!! tip "Open vragen"

    For open questions, you earn points for structure. Do not start with random details. Start with a short definition, then explain the mechanism, give an example, and end with the impact on financial markets or institutions.
