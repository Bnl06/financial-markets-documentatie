# Exam checklist

## Learn first when time is limited

- [ ] Exam analysis and sample-exam analysis read.
- [ ] All **high-priority exam terms** can be defined.
- [ ] Fisher, simple interest, compounding, PV, annuity, bond price/YTM can be applied.
- [ ] Market order, limit order, stop-loss, index weighting and active/passive funds can be compared.
- [ ] Portfolio risk with correlation can be calculated.
- [ ] Forward/future payoff and options payoff/profit/break-even can be explained.
- [ ] Repo, collateral, rating, covenant, negative pledge and ABS are clear.
- [ ] Banking, arranger, syndicated loan, shadow banking and the 2008 crisis can be linked.
- [ ] Regulation tools: deposit insurance, capital adequacy, liquidity requirements, recovery plans.

## Open-question answer check

- [ ] Definition is in the first sentence.
- [ ] Mechanism is explained.
- [ ] There is an example or application.
- [ ] There is a link with a market/institution/crisis/regulation.
- [ ] Answer is concise and not vague.

!!! warning "Check with the official course documents"
    Exact film questions or practical exam settings should be checked in Blackboard/slides.
