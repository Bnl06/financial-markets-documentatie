# Glossary

| Term | Meaning |
|---|---|
| Asset | A possession that has value in an exchange transaction |
| Liability | Debt or obligation |
| Net wealth | Assets minus liabilities |
| Equity | Own funds; the shareholders' residual claim |
| Debt | Borrowed funds; a contractual obligation to pay |
| Stock / share | Ownership claim in a company |
| Bond | Debt security issued by a government or corporation |
| Coupon | Periodic interest payment on a bond |
| Face value | Amount repaid at maturity |
| Present value | Value today of a future cash flow |
| Future value | Value after interest has accrued |
| Yield to maturity | The single discount rate that equates a bond's price with all future cash flows |
| Spot rate | Interest rate for one specific maturity |
| Money market | Market for short-term financing, often below one year |
| Capital market | Market for longer-term financing |
| Primary market | Market where new securities are issued |
| Secondary market | Market where existing securities are traded |
| Leverage | Use of borrowed money to control more assets |
| Gearing | Debt/equity measure of leverage |
| ROA | Return on assets |
| ROE | Return on equity |
| Derivative | Contract whose value depends on an underlying asset or variable |
| Forward | OTC agreement to buy or sell in the future at a fixed price |
| Future | Standardized exchange-traded version of a forward contract |
| Swap | Exchange of cash-flow streams between counterparties |
| Call option | Right to buy the underlying |
| Put option | Right to sell the underlying |
| Premium | Price paid by the option buyer |
| Strike price / exercise price | Agreed price at which the option can be exercised |
| Hedging | Reducing an existing risk exposure |
| Speculation | Taking risk in the hope of making a profit |
| Securitization | Pooling loans/cash flows and turning them into tradable securities |
| MBS | Mortgage-backed security |
| Liquidity risk | Risk of not being able to meet obligations on time |
| Credit risk | Risk that a counterparty does not pay |
| Market risk | Risk caused by market price movements |
| Operational risk | Risk caused by people, processes, systems or errors |
| Systemic risk | Risk that problems spread to the whole financial system |
| Bail-out | Rescue through government/central-bank support, capital or liquidity |
| Moral hazard | Taking more risk because one expects to be rescued |
| SSM | Single Supervisory Mechanism |
| SRM | Single Resolution Mechanism |
| TARP | Troubled Asset Relief Program |
