# Glossary

| Term | Meaning |
|---|---|
| Asset | Bezit met waarde in ruiltransactie |
| Liability | Schuld/verplichting |
| Net wealth | Assets minus liabilities |
| Equity | Eigen vermogen; sharehoudersclaim |
| Debt | Vreemd vermogen; contractuele schuld |
| Stock | Aandeel, eigendomsclaim |
| Bond | Obligatie, lening uitgegeven door overheid/bedrijf |
| Coupon | Periodieke interestbetaling op bond |
| Face value | Bedrag terugbetaald op maturity |
| Present value | Huidige waarde van toekomstige cashflow |
| Future value | Toekomstige waarde na interest |
| Yield to maturity | Discontovoet die bond price gelijk maakt aan cashflows |
| Spot rate | Rente voor één specifieke maturity |
| Money market | Markt voor kortlopende financiering |
| Capital market | Markt voor langlopende financiering |
| Primary market | Nieuwe securities worden uitgegeven |
| Secondary market | Bestaande securities worden verhandeld |
| Leverage | Gebruik van geleend geld |
| Gearing | Debt/equity maatstaf |
| ROA | Return on assets |
| ROE | Return on equity |
| Derivative | Contract afhankelijk van underlying |
| Forward | OTC afspraak voor toekomstige koop/verkoop |
| Future | Gestandaardiseerde beursversie van forward |
| Swap | Uitwisseling van cashflows |
| Call option | Recht om te kopen |
| Put option | Recht om te verkopen |
| Premium | Prijs betaald voor optie |
| Strike price | Uitoefenprijs van optie |
| Hedging | Risico verminderen |
| Speculation | Risico nemen voor mogelijke winst |
| Securitization | Bundelen en verhandelbaar maken van cashflows/leningen |
| MBS | Mortgage-backed security |
| Liquidity risk | Risico niet tijdig te kunnen betalen |
| Credit risk | Risico dat tegenpartij niet betaalt |
| Market risk | Risico door prijsbewegingen |
| Operational risk | Risico door fouten/systemen/processen |
| Systemic risk | Risico voor het hele systeem |
| Bail-out | Redding met steun/kapitaal/liquiditeit |
| Moral hazard | Meer risk nemen omdat men redding verwacht |
| SSM | Single Supervisory Mechanism |
| SRM | Single Resolution Mechanism |
| TARP | Troubled Asset Relief Program |
